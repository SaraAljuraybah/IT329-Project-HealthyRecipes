# IT329-Project-HealthyRecipes

A healthy **lunch box recipe sharing** web application built for IT329 – Advanced Web Technologies (KSU).
The platform helps users discover, share, and manage lunchbox-friendly recipes (quick, nutritious, and easy to pack).

## Concept
**LunchBoxLab** focuses on recipes designed for:
- School lunch boxes
- Work lunch boxes
- Meal-prep / on-the-go meals

### Example Categories (choose at least 3)
- Kids Lunch Boxes
- Office Lunch Boxes
- High-Protein Boxes
- Low-Calorie Boxes
- Vegetarian / Vegan Boxes
- Gluten-Free Boxes
- No-Heat / Cold Boxes

## Users & Features
### Regular Users
- View all recipes
- Like recipes (Phase 1: UI only)
- Add recipes to favourites (Phase 1: UI only)
- Comment on recipes (Phase 1: UI only)
- Report inappropriate recipes (Phase 1: UI only)
- Add/edit/delete their own recipes (delete is UI-only in Phase 1)

### Admins
- Review reported recipes
- Block users (Phase 1: UI only)
- Dismiss reports (Phase 1: UI only)
